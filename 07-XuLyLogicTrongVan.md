# 07 - Xử Lý Logic Trong Ván\n\n## 🎯 Tổng Quan Xử Lý Logic\n\nKhi một ván cờ đang diễn ra, nhiều thành phần phải hoạt động đồng thời:\n\n```\nAppController.Run()\n    ├─→ Input Handler\n    │   └─→ Nhập nước đi → MoveParser → Move object\n    │\n    ├─→ Move Validator\n    │   ├─→ Xác thực hợp lệ\n    │   └─→ Kiểm tra không để vua bị chiếu\n    │\n    ├─→ Board & History Manager\n    │   ├─→ Thực hiện nước (MakeMove)\n    │   ├─→ Lưu vào history (HistoryManager)\n    │   └─→ Cập nhật cây (BranchTracker)\n    │\n    ├─→ Game Clock\n    │   ├─→ Tính thời gian\n    │   ├─→ Trừ & cộng increment\n    │   └─→ Kiểm tra hết giờ\n    │\n    └─→ Move Validator (lại)\n        ├─→ Kiểm tra checkmate\n        └─→ Kiểm tra stalemate\n```\n\n---\n\n## 🔍 Xác Thực Nước Đi Chi Tiết\n\n### Bước 1: Xác Thực Cơ Bản\n\n```csharp\nIMoveValidator.IsValidMove(board, move, currentTurn)\n    ↓\n1. Kiểm tra vị trí:\n   ├─→ move.From trong giới hạn? board.IsValidPos(move.From)\n   ├─→ move.To trong giới hạn? board.IsValidPos(move.To)\n   └─→ Nếu NO: return false\n\n2. Kiểm tra quân:\n   ├─→ GetPieceAt(move.From) != null?\n   ├─→ piece.Color == currentTurn?\n   ├─→ GetPieceAt(move.To) == null OR piece.Color != currentTurn?\n   └─→ Nếu NO: return false\n\n3. Kiểm tra quân có thể di chuyển:\n   ├─→ piece.GetValidMoves(board, move.From)\n   ├─→ move.To in validMoves?\n   └─→ Nếu NO: return false\n```\n\n### Bước 2: Xử Lý Nước Đặc Biệt\n\n#### **Cờ Vua:**\n\n```csharp\n// En Passant (ăn tây tạp)\nif (piece.Type == Pawn && move.IsDiagonal) {\n    ├─→ board.LastMove.To.Row == move.From.Row?\n    ├─→ board.LastMove.To.Col == move.To.Col?\n    ├─→ board.LastMove.Piece.Type == Pawn?\n    ├─→ board.LastMove.WasTwoSquareMove()?\n    ├─→ Nếu YES: move.IsEnPassant = true\n    └─→ move.CapturedPiece = quân tại (move.From.Row, move.To.Col)\n}\n\n// Castling (nhập thành)\nif (piece.Type == King) {\n    ├─→ move.FromCol == 4? (e-file cho White)\n    ├─→ Rook trước đó bị di chuyển? (kiểm tra _pieceMoved)\n    ├─→ King trước đó bị di chuyển? (kiểm tra _pieceMoved)\n    ├─→ Nếu move.ToCol == 2 (Queenside): Castling trái\n    │   ├─→ move.IsCastling = true\n    │   ├─→ CastlingRookMove = từ (row, 0) → (row, 3)\n    │   └─→ Kiểm tra ô giữa trống: (row, 1-3)\n    ├─→ Nếu move.ToCol == 6 (Kingside): Castling phải\n    │   ├─→ move.IsCastling = true\n    │   ├─→ CastlingRookMove = từ (row, 7) → (row, 5)\n    │   └─→ Kiểm tra ô giữa trống: (row, 5-6)\n    └─→ Kiểm tra King không ở check\n}\n\n// Promotion (phong cấp)\nif (piece.Type == Pawn) {\n    ├─→ (White && move.To.Row == 0) || (Black && move.To.Row == 7)?\n    ├─→ move.PromotionPiece != null?\n    ├─→ move.PromotionPiece in [Queen, Rook, Bishop, Knight]?\n    └─→ Nếu NO: return false\n}\n```\n\n#### **Cờ Tướng:**\n\n```csharp\n// General giới hạn cung điện\nif (piece.Type == General) {\n    ├─→ IsWithinPalace(move.To, piece.Color)?\n    └─→ Nếu NO: return false\n}\n\n// Advisor giới hạn cung điện + chéo\nif (piece.Type == Advisor) {\n    ├─→ IsWithinPalace(move.To, piece.Color)?\n    ├─→ IsDiagonal(move)?\n    ├─→ Distance == 1?\n    └─→ Nếu NO: return false\n}\n\n// Elephant không qua sông + trung điểm\nif (piece.Type == Elephant) {\n    ├─→ IsCrossingSide(move)?\n    │   └─→ Nếu YES: return false (qua sông)\n    ├─→ IsBlockedAtMidpoint(move)?\n    │   └─→ Nếu YES: return false (bị chặn)\n    └─→ OK\n}\n\n// Horse bị chặn hàng\nif (piece.Type == Horse) {\n    ├─→ blockingPos = GetAdjacentPos(move.From, move.To)\n    ├─→ GetPieceAt(blockingPos) != null?\n    │   └─→ Nếu YES: return false (bị chặn)\n    └─→ OK\n}\n\n// Soldier quy tắc trước/sau sông\nif (piece.Type == Soldier) {\n    ├─→ HasCrossedRiver(piece)?\n    ├─→ Nếu NO:\n    │   ├─→ Chỉ di chuyển tiến hoặc ngang\n    │   └─→ IsMoveForwardOrSide(move)?\n    └─→ Nếu YES:\n        ├─→ Di chuyển 1 ô theo 4 hướng\n        └─→ CanMove4Directions(move)?\n}\n\n// Cannon không ăn phải trống, ăn phải có trung điểm\nif (piece.Type == Cannon) {\n    ├─→ Nếu KHÔNG ăn:\n    │   ├─→ IsPathClear(move)?\n    │   └─→ Nếu NO: return false\n    └─→ Nếu ĂN:\n        ├─→ blockingPieces = CountBlockingPieces(move)\n        ├─→ blockingPieces.Count == 1?\n        └─→ Nếu NO: return false\n}\n```\n\n### Bước 3: Kiểm Tra An Toàn Vua\n\n```csharp\nMoveLeavesKingInCheck(board, move, currentTurn):\n    ├─→ clone = board.Clone()\n    ├─→ clone.MakeMove(move)\n    ├─→ IsCheck(clone, currentTurn)?\n    ├─→ Nếu YES: return true (vua bị chiếu)\n    └─→ Nếu NO: return false (an toàn)\n```\n\n---\n\n## ⏱️ Quản Lý Thời Gian (GameClock)\n\n### Khởi Tạo\n\n```csharp\nnew GameClock(ClockSettings settings)\n    ├─→ _settings = settings\n    ├─→ TimeWhiteSeconds = settings.InitialTimeSeconds\n    ├─→ TimeBlackSeconds = settings.InitialTimeSeconds\n    ├─→ _currentPlayer = Color.None\n    ├─→ _turnStartTime = null\n    ├─→ _timer = new Timer(1000) // 1 giây\n    └─→ _timer.Elapsed += OnTimerTick\n```\n\n### Chu Kỳ Một Lượt\n\n```\nKhi nước trắng bắt đầu:\n├─→ clock.StartTurn(Color.White)\n│   ├─→ _currentPlayer = White\n│   ├─→ _turnStartTime = DateTime.UtcNow\n│   └─→ _timer.Start()\n│\n├─→ [Mỗi 1 giây: OnTimerTick]\n│   ├─→ elapsed = (DateTime.UtcNow - _turnStartTime).TotalSeconds\n│   ├─→ OnTimeUpdated event (UI cập nhật)\n│   └─→ Nếu elapsed > TimeWhiteSeconds:\n│       ├─→ OnTimeOut event\n│       └─→ _timer.Stop()\n│\n└─→ Khi trắng kết thúc nước:\n    └─→ clock.StopTurn()\n        ├─→ elapsed = (DateTime.UtcNow - _turnStartTime).TotalSeconds\n        ├─→ TimeWhiteSeconds -= (int)elapsed\n        ├─→ TimeWhiteSeconds += _settings.IncrementSeconds\n        ├─→ _timer.Stop()\n        ├─→ OnTurnStopped event\n        └─→ Sẵn sàng cho StartTurn(Color.Black)\n```\n\n### Ví Dụ Timeline\n\n```\nBlitz: 5 phút (300 giây) + 0 increment\n\n[0:00] White StartTurn\n[0:05] White thực hiện nước\n       ├─→ Elapsed: 5 giây\n       ├─→ TimeWhiteSeconds: 300 - 5 = 295 giây\n       ├─→ Increment: +0 = 295 giây\n       └─→ StopTurn\n\n[0:05] Black StartTurn\n[0:10] Black thực hiện nước  \n       ├─→ Elapsed: 5 giây\n       ├─→ TimeBlackSeconds: 300 - 5 = 295 giây\n       ├─→ Increment: +0 = 295 giây\n       └─→ StopTurn\n\n[0:10] White StartTurn\n...\n\nRapid: 10 phút (600 giây) + 3 increment\n\n[0:00] White StartTurn\n[0:05] White thực hiện nước\n       ├─→ Elapsed: 5 giây\n       ├─→ TimeWhiteSeconds: 600 - 5 = 595 giây\n       ├─→ Increment: +3 = 598 giây\n       └─→ StopTurn\n\n[0:05] Black StartTurn\n...\n```\n\n---\n\n## 📜 Quản Lý Lịch Sử (HistoryManager)\n\n### Cấu Trúc Command\n\n```csharp\ninterface ICommand:\n    ├─→ Execute(): Thực hiện nước\n    └─→ Undo(): Hoàn tác nước\n\nclass MoveCommand implements ICommand:\n    ├─→ _board: Bàn cờ\n    ├─→ _move: Nước đi\n    ├─→ _previousBoardState: Clone bàn trước\n    │\n    ├─→ Execute():\n    │   ├─→ _previousBoardState = _board.Clone()\n    │   ├─→ _board.MakeMove(_move)\n    │   └─→ Cập nhật _move.San nếu cần\n    │\n    └─→ Undo():\n        ├─→ _board = _previousBoardState.Clone()\n        └─→ Khôi phục state trước\n```\n\n### Stack Operations\n\n```\nBan đầu:\n_undoStack = []\n_redoStack = []\n\nSau 3 nước:\n_undoStack = [Cmd1, Cmd2, Cmd3]\n_redoStack = []\n\nUndo 1 lần:\n_undoStack = [Cmd1, Cmd2]\n_redoStack = [Cmd3]\n\nUndo 2 lần:\n_undoStack = [Cmd1]\n_redoStack = [Cmd3, Cmd2]\n\nRedo 1 lần:\n_undoStack = [Cmd1, Cmd2]\n_redoStack = [Cmd3]\n\nThực hiện nước mới:\n_undoStack = [Cmd1, Cmd2, Cmd4]\n_redoStack = [] (CLEAR!)\n```\n\n---\n\n## 🌳 Quản Lý Nhánh (BranchTracker)\n\n### Tree Structure\n\n```\n        Root\n         │\n    [Move e4]\n         │\n    [Move c5] ──── [Move e5] (nhánh thay thế)\n         │\n    [Move Nf3]\n         │\n    [Move d6] ──── [Move e6] (nhánh thay thế)\n                ── [Move a6] (nhánh khác)\n         │\n    [Move d4]\n```\n\n### Các Phương Thức\n\n```csharp\nAddMove(move):\n    ├─→ Check if move in _currentNode.Children\n    ├─→ Nếu YES: _currentNode = node có move\n    ├─→ Nếu NO:\n    │   ├─→ newNode = new Node(move, _currentNode)\n    │   ├─→ _currentNode.Children.Add(newNode)\n    │   └─→ _currentNode = newNode\n    └─→ _mainLine.Add(move)\n\nGoBack():\n    ├─→ Nếu _currentNode.Parent != null:\n    │   ├─→ _currentNode = _currentNode.Parent\n    │   ├─→ _mainLine.RemoveAt(Count - 1)\n    │   └─→ return true\n    └─→ Else: return false\n\nGetCurrentLine():\n    ├─→ Duyệt từ _currentNode ngược về root\n    └─→ return danh sách nước từ root → hiện tại\n\nGetAvailableBranches():\n    └─→ return _currentNode.Children.Select(n => n.Move)\n\nGoToMoveIndex(index):\n    ├─→ Duyệt từ root → nước thứ index\n    ├─→ _currentNode = node tại vị trí\n    └─→ _mainLine = danh sách đến vị trí\n```\n\n---\n\n## 🏁 Kiểm Tra Kết Thúc Ván\n\n### Checkmate / Chiếu Hết\n\n```csharp\nIsCheckmate(board, color):\n    ├─→ // 1. Kiểm tra vua bị chiếu\n    ├─→ kingPos = FindKing(board, color)\n    ├─→ IsKingUnderAttack(board, kingPos)?\n    │   └─→ Nếu NO: return false (không checkmate nếu không chiếu)\n    │\n    ├─→ // 2. Kiểm tra không có nước thoát\n    ├─→ allMoves = GetAllValidMoves(board, color)\n    ├─→ allMoves.Count == 0?\n    │   └─→ Nếu NO: return false (còn nước thoát)\n    │\n    └─→ Nếu cả 2 đúng: return true (CHECKMATE)\n\nGetAllValidMoves(board, color):\n    ├─→ validMoves = []\n    ├─→ Với mỗi piece của color:\n    │   ├─→ basicMoves = piece.GetValidMoves(board, pos)\n    │   ├─→ Với mỗi move trong basicMoves:\n    │   │   ├─→ clone = board.Clone()\n    │   │   ├─→ clone.MakeMove(move)\n    │   │   ├─→ IsCheck(clone, color)? NO\n    │   │   ├─→ Nếu an toàn: Add move vào validMoves\n    │   │   └─→ Nếu không: Skip\n    │   └─→ (Xong loop)\n    └─→ return validMoves\n```\n\n### Stalemate / Hòa Cờ\n\n```csharp\nIsStalemate(board, color):\n    ├─→ // 1. Kiểm tra vua KHÔNG bị chiếu\n    ├─→ kingPos = FindKing(board, color)\n    ├─→ IsKingUnderAttack(board, kingPos)?\n    │   └─→ Nếu YES: return false (checkmate hoặc other, không stalemate)\n    │\n    ├─→ // 2. Kiểm tra không có nước hợp lệ\n    ├─→ allMoves = GetAllValidMoves(board, color)\n    ├─→ allMoves.Count == 0?\n    │   └─→ Nếu NO: return false (còn nước đi)\n    │\n    └─→ Nếu cả 2 điều kiện: return true (STALEMATE)\n```\n\n### Ví Dụ\n\n```\nScenario 1: Checkmate\n├─→ Vua trắng tại e1, bị hậu đen tấn công từ e4\n├─→ Trắng kiểm tra:\n│   ├─→ IsCheck? YES (hậu tấn công e1)\n│   ├─→ GetAllValidMoves? [] (không có nước)\n│   │   ├─→ Vua không thể di chuyển (d1, d2, e2, f1, f2 đều bị tấn công)\n│   │   ├─→ Không thể chặn hậu\n│   │   └─→ Không thể ăn hậu\n│   └─→ CHECKMATE\n\nScenario 2: Stalemate\n├─→ Vua trắng tại a1, NO quân khác\n├─→ KHÔNG bị chiếu\n├─→ Trắng kiểm tra:\n│   ├─→ IsCheck? NO\n│   ├─→ GetAllValidMoves? [] (vua chỉ có thể đi b1, a2, b2)\n│   │   ├─→ b1 bị tấn công\n│   │   ├─→ a2 bị tấn công\n│   │   └─→ b2 bị tấn công\n│   └─→ STALEMATE\n```\n\n---\n\n**Tiếp theo:** [08-LuồngReplay.md](08-LuồngReplay.md) - Luồng xem lại ván\n
